关于后台管理应用

# 名词约定

  • Dashboard 控制台/控制面板
  • admin 管理
  • permssim 权限
  • 接口（前后端之间的接口，一般指url）

# 单页/多页

  • 单页：用router接管所有页面的路径，访问的都是同一个页面。无SEO
  • 多页：每个页面都是不同的html文件，访问的都是不同页面。利于SEO

# RESTful API——URL起名字方法论

  • 登录
  • 登出
  • 注册
  • 购买

# 权限设置

  • 权限跟角色分离，也就是说管理员1和管理员2虽然同是管理员但是可以分配不同权限。
  • 权限跟角色耦合，也就是是管理员1跟管理员2的权限是一样的，1有怎样的权限2就有。
  • 根据数据写个配置页面，让超级管理员配置哪些角色能看哪些菜单。
