# 远程办公的自我修养
今年远程办公满一年了，总结一些心得体会。

## 如何推进工作
实话说，远程工作确实对每个人的职业素养是一种考验。要想有效推进项目，以下几点至关重要：
- 自我驱动：每个人都应该成为任务的推进器，而不是需要领导不断催促才能前进。如果依赖领导的推动，项目很难有实质性的进展。
- 减少返工：在远程工作中，来回返工会非常耗时。因此，开发过程中的单元测试、自我检查和对需求的深刻理解显得尤为重要。
- 及时沟通：保持沟通的及时性。如果是异步沟通（比如不同时区），我们可以另开一篇讨论。
- 合理使用工具：虽然市面上有很多工作推进工具，我不特别推荐具体哪一款，因为我更想说明的是从人出发去管理，工具来做辅助，而且很多工具设计本身就不合理，如果不了解其本质可能会反噬，例如每日没营养的例会，每日汇报进度，每人/时间/产出 这种形式主义只会浪费时间，remote本质是让每个人的时间更值钱，而不是让你的时间变成垃圾。

## 需要什么技能
- 独立解决问题的能力：这是远程工作中最重要的能力之一。你不能随时向身边的同事求助，因此需要强大的独立解决问题的能力。
- 沟通能力：在远程工作中，准确、高效地沟通需求和问题至关重要。能否在短时间内明确表达问题点，直接影响工作效率。

## 带来什么改变
首先，你的可支配时间增加了。如何合理安排这些时间至关重要。没有计划的生活可能会导致你睡到中午，作息不规律。这就像不懂如何管理财富的人一样，财富反而会控制了他们。因此，你需要设定清晰的日常计划，确保每天按时起床，处理工作。我在远程工作初期也经历过一段非常难受的适应期。我以为自己会更自由，但事实上，缺乏约束让我更加放纵。我经常深夜睡觉，中午起床，导致生物钟紊乱，思维迟钝。后来，我制定了早起计划，并用闹钟来提醒自己每天的任务，从而逐渐适应了远程工作。
