[**Element**](https://developer.mozilla.org/zh-CN/docs/Web/API/Element)是非常通用的基类，所有 Document对象下的对象都继承它. 这个接口描述了所有相同种类的元素所普遍具有的方法和属性。

**Element.getBoundingClientRect()**
方法返回元素的大小及其相对于视口的位置。
