## 注意事项

SVG的元素和属性必须按标准格式书写，因为XML是区分大小写的，属性值必须用引号引起来，就算是数值也必须这样做。

## 属性

- viewBox 对内部局部放大缩小
- x 横轴，y 纵轴
- width 宽度 height 高度
- fill 内部填充色
- stroke-width='5' 边框宽度

## 标签

- rect 矩形
  - rx ,ry 圆角属性
