## 变量提升（hoisting）
```javascript
console.log(num)	//undefined
var num=123
```
总结：变量提升只存在var变量声明中，这个是var其中一个副作用，所以在es6出现时推介使用代替另外var另一个副作用是只有函数作用域，没有块级作用域。
