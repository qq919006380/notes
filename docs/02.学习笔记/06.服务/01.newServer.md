### 下载tmux
```bash
sudo apt-get install tmux
```

### 使用
```bash
# 进入tmux
$ tmux 
# 唤起快捷键
$ ctrl + b

Ctrl+b d：分离当前会话。(挂起并退出)
Ctrl+b s：列出所有会话。
Ctrl+b $：重命名当前会话。
```
