## 请求用户签名
```js
    const message = 'Hello, world!';
    try {
      const signature = await ethereum.request({
        method: 'personal_sign',
        params: [message, acc],
      });
      console.log('Signature:', signature);
    } catch (error) {
      console.error('Error signing message:', error);
    }
```

## Call
call用来调用view和pure函数，它只运行在本地节点，不会在区块链上创建事务

## Send
send将创建一个事务并改变区块链上的数据，你需要使用send来调用任何非view或者pure的函数，注意：send一个事务将要求用户支付gas
